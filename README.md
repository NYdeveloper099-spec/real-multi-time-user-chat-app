# Real-Time Chat App

A full-stack real-multi-time-user-chat application built with:

- React
- Node.js
- Express
- MongoDB
- Socket.io
- JWT Authentication
- Cloudinary Image Upload

## Features

- User Registration
- Login Authentication
- Avatar Upload
- Real-Time Messaging
- Message Alignment
- Protected Routes
- MongoDB Persistence

## Future Improvements

- Online Status
- Typing Indicators
- Read Receipts
- Private Conversations
- Group Chats
